# TechFest 2026 

TechFest 2026 is a multi-page website created using HTML. The project represents a college technology festival and includes information about events, registration, sponsors, and contact details.

## Features

* Home page with event introduction
* Event schedule page
* Registration form
* Sponsors page
* Contact page
* Google Maps integration using iframe
* Embedded YouTube video
* Navigation between multiple pages
* Responsive viewport setup

## Technologies Used

* HTML5
* Google Maps Embed
* YouTube Embed

## Project Structure

* index.html – Home Page
* events.html – Event Schedule
* registration.html – Registration Form
* sponsors.html – Sponsors Information
* contact.html – Contact Page
* techfest.png – Website Banner Image

## Purpose

This project was built as a beginner web development project to practice HTML concepts such as:

* Forms
* Tables
* Hyperlinks
* Images
* Lists
* Iframes
* Semantic HTML Elements
